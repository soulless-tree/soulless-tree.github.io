It took some time, but after the release of this really helpful Luke Smith’s tutorial on Hugo I was finally able to port my old site to this framework. I must say that, although most of the process was an easy rewriting of what I already had, the seemingly simple navigation bar was by far the most laborious thing to make. First, because I had to figure out how to add non-section Markdown files (the Home and About sections) to the Section Menu for Lazy Bloggers, which later I learned had to be added manually in the config file as follows:

[menu]
[[menu.main]]
name = 'Home'
url = '/'

[[menu.main]]
name = 'About'
url = '/about'

The second reason was that I couldn’t mantain the highlighting of the selected menu item within section posts, but luckily I found a solution. And now that everything is working properly I can finally look foward to implement some kind of interactive gallery to the Art section (maybe this is the time to learn JavaScript).
